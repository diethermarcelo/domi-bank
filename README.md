#  domi-bank
- A command line interface app for bank account transactions. 

## Features
- Authentication (Login/Register/Change Money)
- Withdraw
- Deposit
- Check balance
- Transfer Money
- Users
- Business Logic
    - Register only if there's money

## Technologies
- C++ 12.2
- G++ Compiler

- 

## Run
- g++ -o build/main src/main.cpp
- cd build && main.exe
